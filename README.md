# Simple To-Do CLI

A simple, terminal-based To-Do application written in Python.

## Features

- **Add Task**: Quickly add new tasks with descriptions.
- **Edit Task**: Modify existing tasks and update their status (Pending/Done).
- **Remove Task**: Delete tasks from your list.
- **Tabular Display**: View your tasks in a clean, formatted table.
- **Persistence**: Your tasks are saved automatically to a `tasks.json` file.

## How to Run

1. Make sure you have Python installed on your system.
2. Navigate to the project directory in your terminal.
3. Run the following command:
   ```bash
   python todo.py
   ```

## Usage

- Follow the on-screen menu to interact with the application.
- Enter the number corresponding to the action you want to perform.
- Use Task IDs to select tasks for editing or removal.
